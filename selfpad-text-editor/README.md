# Custom Text Editor (Notepad Clone)

A simple and lightweight Text Editor application built using Python, inspired by Windows Notepad.

## 📌 About The Project

This project was built as the **final capstone project** for completing a comprehensive Python 3 learning course. It is designed to practice core Python concepts, GUI building, and file handling.

* **Project Type:** Practice / Learning Project
* **Course:** Python 3.5 Course
* **Instructor / Channel:** Harshit Vashisth

## ✨ Features

* Text editing capabilities similar to Windows Notepad
* Create, open, edit, and save text files
* Simple and clean user interface

## 🛠️ Built With

* **Python 3**
* **Tkinter** *(Python's standard GUI framework)*
