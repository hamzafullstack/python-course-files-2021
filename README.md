## 📁 Repository Structure & Contents

Here is the breakdown of all the topic folders included in this repository:

| Folder | Description / Key Topics |
| :--- | :--- |
| `01-basics` | Python syntax, print statements, and fundamental concepts |
| `02-variables` | Declaring variables, data types, and basic operations |
| `03-conditionals-if-else` | Control flow using `if`, `elif`, and `else` statements |
| `04-functions` | Defining and calling functions, arguments, and return values |
| `05-lists` | Working with Python lists (indexing, slicing, methods) |
| `06-tuples` | Immutable sequences and tuple operations |
| `07-dictionaries` | Key-value data structures and dictionary methods |
| `08-sets` | Unique elements collection and set mathematical operations |
| `09-lists-comprehension` | Elegant and concise list creation methods |
| `10-dictionary-comprehension` | Concise syntax for building dictionaries |
| `11-args` | Flexible function arguments (`*args` and `**kwargs`) |
| `12-lambda` | Anonymous inline functions |
| `13-enumerate-functions` | Iterating with index-value pairs using `enumerate()` |
| `14-decorators` | Function wrappers, nested functions, and metaprogramming |
| `15-generators` | Memory-efficient iteration using `yield` |
| `16-oop` | Object-Oriented Programming (Classes, Inheritance, Polymorphism) |
| `17-error-handling` | Exception handling using `try`, `except`, `finally` |
| `18-read-write-files` | File I/O operations (creating, reading, writing text files) |
| `19-read-files` | Advanced file reading techniques |
| `20-os-module` | System interactions, file pathways, and directory navigation |
| `21-projects` | Basic practice projects and mini-tasks |
| `selfpad-text-editor` | **⭐ Final Capstone Project:** A complete Notepad-like Text Editor GUI |

## 🛠️ Final Capstone Project: Selfpad Text Editor

The **`selfpad-text-editor`** folder contains the final capstone project built using Python and Tkinter.

### Features:
* Text editing capabilities similar to Windows Notepad
* Create, open, edit, and save `.txt` files
* Built with Python 3.5 & Tkinter
